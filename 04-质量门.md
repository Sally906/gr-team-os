# 四、质量门（Quality Gate）

**所有对外GR输出必须经过质量门**。未通过不得发出。

## 检查清单

| 检查项 | 具体问题 | 结果 |
| :--- | :--- | :--- |
| 来源 | 引用的数据、政策条款是否有可查证来源？来源是否最新？ | 是/否/NA |
| 事实 | 公司名称、数据、日期、人名、职务是否准确？ | 是/否/NA |
| 合规 | 是否涉及未公开信息、敏感数据？是否合规？ | 是/否/NA |
| 业务逻辑 | 诉求、建议是否符合公司实际能力？ | 是/否/NA |
| 语气 | 语气是否专业、尊重、立场正确？ | 是/否/NA |
| 可执行性 | 行动项是否有负责人、截止时间、成功标准？ | 是/否/NA |
| 审批状态 | 是否已获得DRI或Leader审批？ | 是/否/NA |

**通过标准**：所有适用项均为“是”。任一“否”则退回修改并注明原因。
